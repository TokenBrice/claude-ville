# Support

ClaudeVille is a local desktop dashboard. Start with the matching runbook before opening an issue.

| Need | Start here |
| --- | --- |
| First-hour setup, provider, API, or graphics issue | `docs/troubleshooting.md` |
| Provider paths and supported session stores | `README.md` Provider Adapters section |
| Add or change a provider/model identity | `docs/agent-provider-addition.md` |
| World mode visual QA | `docs/world-visual-qa-checklist.md` |

Use GitHub Issues for reproducible bugs, provider parsing problems, docs problems, and focused feature requests. Use GitHub Discussions for broad ideas, roadmap questions, and new provider proposals.

Do not post provider logs, tokens, API keys, private file paths, or screenshots with secrets in public issues.
