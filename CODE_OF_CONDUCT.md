# Code Of Conduct

ClaudeVille welcomes focused, respectful collaboration.

## Expected Behavior

- Be direct, constructive, and specific.
- Assume local-first privacy and maintainer time matter.
- Keep discussions on the technical issue, user workflow, or project goal.
- Redact private paths, logs, tokens, and screenshots before posting.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing another person's private data, credentials, logs, or session content.
- Repeated off-topic demands after maintainers have clarified scope.
- Security disclosure in public issues before a mitigation is available.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; restrict participation; or block users when behavior harms the project or community. Security-sensitive reports should follow `SECURITY.md`.
